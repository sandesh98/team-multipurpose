Wat is Bootstrap?

En wat heb jij eraan als beginnend designer?

Bootstrap is een gratis CSS-framework dat een basiscollectie van de meest gebruikte webcomponenten bevat. Het wordt door veel beginnende web developers gebruikt omdat de documentatie uitgebreid is en veel voorbeelden bevat die gekopieerd en geplakt kunnen worden in je eigen project, of simpel weg om uit te proberen.

Daarbij is Bootstrap ook een tool waarmee je gemakkelijk een prototype tot leven kan brengen. Je kan er gemakkelijk websites mee maken, zoals een portfolio website of one page website (zoals deze), zonder al te veel eigen CSS voor te hoeven schrijven. Op het internet zijn ook veel gratis templates te vinden waarin jij alleen nog de teksten hoeft aan te passen. Bootstrap biedt ook veel zelfgemaakte en gratis templates aan die als start van een website kunnen dienen. Deze websites bestaan uit simpele portfolio websites tot volledige dashboards websites.

Het framework maakt gebruik van een 12-grid kolommen systeem. Dit is een standaard voor veel van de bestaande websites zoals Spotify en Zoom. En designers hanteren het 12-grid kolommen systeem bij het ontwerpen van responsieve ontwerpen.

Hoe zet je Bootstrap in?

Wat is er nodig om Bootstrap te kunnen gebruiken?

Bootstrap kan op veel verschillende manieren binnen een project gehaald worden. De twee bekendste manieren zijn via NPM en via een CDN. Een CDN is de makkelijkste manier aangezien dit niet heel anders is dan een normaal CSS-bestand in een HTML-document verwerken. En direct nadat de basisstructuur van de HTML-document opgezet is heb je de mogelijkheid om componenten uit de documentatie te kopiëren en plakken.

Bootstrap kan in een vroeg stadium beschouwd worden als een vertelvorm. Met een vroeg stadium bedoelen wij wanneer je net begint met het framework. Een voorbeeld van een vertelvorm is een one page website met alle inzichten van een onderzoek in plaats van een traditionele presentatie. Ook kan het een vertelvorm zijn in de vorm van een portfolio website waarin jij je zelf presenteert naar de buitenwereld. Natuurlijk kan het verder gaan dan dat. Websites zoals spotify, Americanexpress en Salesforce zijn ontworpen in Bootstrap. De websites lijken uiteraard niet op elkaar omdat zij een eigen Design Guideline en Rules hanteren. Design Guidelines en Rules omschrijven hoe componenten samenkomen en hoe de typografie en kleuren met elkaar gebruikt mogen worden. Dit laat zien hoe krachtig het framework kan zijn wanneer er een set aan regels opgesteld is. Hieronder worden nog een paar punten uitgelicht waarom Bootstrap populair is.

Waarom Bootstrap?

Waarom zou jij Bootstrap willen gebruiken als beginnend designer?

Voor we dit stuk schreven hebben we veel onderzoek gedaan naar Bootstrap. We hebben literaire onderzoeken gelezen maar ook gesproken met een expert. De voornaamste reden waarom Bootstrap ingezet wordt is omdat het de mogelijkheid biedt voor rapid development. Doordat het een goed doordachte Style Guideline bevat biedt het de mogelijkheid om een design te programmeren dat consistent en responsive werkt (responsive houdt in dat de website geschikt is voor alle mogelijke apparaten, dus ook mobiele apparaten zoals telefoons en tablets).

Zoals eerder benoemd biedt Bootstrap gratis web templates aan en staat de documentatie vol met voorbeelden die je kan gebruiken in je eigen website. Het een vertelvorm tijdens een project en we hebben zelfs instanties gezien waarin het Design Challenge verslag gemaakt is in Bootstrap. Over schoolprojecten gesproken, na het maken van een highfidelity prototype kan je het project naar een hoger niveau brengen door het ontwerp daadwerkelijk te realiseren. Het voordeel is dat je ontzettend veel leert van een ontwerp daadwerkelijk programmeren. Daarbij zorgt het voor betere resultaten tijdens het testen van je ontwerp. En zodoende kan je de docenten en opdrachtgever ervan overtuigen dat je ontwerp werkt omdat de doelgroep de dienst echt heeft kunnen ervaren.

Het zelf hosten van een Bootstrap website is een stuk voordeliger dan gebruik maken van een CMS-systeem. Het gebruiken van een CMS, ook wel Content Management Systeem, helpt jou als ontwerper om responsieve websites te maken zonder enige kennis over HTML en CSS te hebben. Eén van de grootste nadelen van een CMS-systeem gebruiken is dat je gemiddeld €14,95 per maand kwijt bent. De grootste CMS-systemen zoals Webflow en Wix accepteren geen iDEAL dus je bent genoodzaakt om ook nog eens een creditcard aan te schaffen als je die nog niet hebt. Zonder een betaalmethode toegevoegd te hebben kan je wel gratis je website opbouwen maar zodra je online wilt gaan dan dien je een betaalmethode toe te voegen.

Klein feitje tussendoor

De meest recente versie van Bootstrap is versie 5

2021 - heden

Bootstrap 5

Introductie van Utilitie classes en nieuwe componenten. Maakt geen gebruik meer van jQuery.

2015 - 2021

Bootstrap 4

Meer focus op toegankelijkheid op alle componenten voor slechtziende mensen op alle componenten.

2013 - 2015

Bootstrap 3

De focus werd zwaarder gelegd op de verschillende scherm formaten: xs - sm - md - lg.

2012 - 2012

Bootstrap 2

Geupdate van versie 1 naar 2 om de nieuwe structuren van HTML 5 te ondersteunen.

2011 - 2012

Bootstrap 1

Ookwel Twitter Bootstrap genoemd. Is gedesigned en ontwikkeld door Twitter
